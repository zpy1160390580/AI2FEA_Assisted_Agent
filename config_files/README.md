# ============================================================================
# 配置文件夹说明
# ============================================================================
#
# 本文件夹包含项目的所有配置文件：
#
# 1. config.py - 主配置文件
#    - 包含所有可配置参数（模型、阈值、UI 等）
#    - 详细的中文注释
#    - 可以提交到 Git
#
# 2. .env.example - 环境变量模板
#    - 仅包含 API Key 配置示例
#    - 可以提交到 Git
#    - 使用方法：复制为 .env 并填入真实 API Key
#
# 3. .env - 实际环境变量文件（用户创建）
#    - 包含真实的 API Key
#    - 不应提交到 Git（已在 .gitignore 中）
#
# ============================================================================
# 使用流程
# ============================================================================
#
# 1. 复制模板文件：
#    cp .env.example .env
#
# 2. 编辑 .env 文件，填入您的 API Key：
#    SILICONFLOW_API_KEY=your_real_api_key_here
#
# 3. 修改其他配置参数（可选）：
#    编辑 config.py 文件
#
# 4. 运行应用：
#    streamlit run main.py
#
# ============================================================================
